## 対応内容

以下の修正を実施済みです。

### 修正済み項目

#### 1) 1.4.10 リフロー — 固定幅によるレスポンシブ対応 → **修正済み**

- **問題**: `w-[448px]` の固定幅により、400%ズーム時や狭い親コンテナで横スクロールが発生する可能性があった
- **対応**: `max-w-full` を追加し、親コンテナより幅が狭い場合も対応

```tsx
// 適用された修正
const inlineMessageVariants = cva(
  "relative w-[448px] max-w-full rounded-md border p-3 flex gap-2 items-start",
  // ...
);
```

## 推奨テストケース

既存のテストで以下を検証済み:

- `role="alert"` が設定されること
- タイトルがある場合、`aria-labelledby` がタイトル要素の ID に設定されること
- 説明がある場合、`aria-describedby` が説明要素の ID に設定されること
- タイトルがない場合、`aria-labelledby` が設定されないこと
- 閉じるボタンに `aria-label="閉じる"` が設定されること
