# sparkle-design 概要
- 目的: Sparkle Design の React コンポーネントライブラリ（shadcn/ui ベース）。
- 技術: React, TypeScript, Next.js, Tailwind CSS, Storybook, Vitest。
- 構成: `src/`（components, app, lib）, `public/`, `scripts/`, `docs/ai-instructions/`。
- パッケージ: `sparkle-design`（dist を exports）。
- 重要ファイル: `sparkle.config.json`, `components.json`, `README.md`。
