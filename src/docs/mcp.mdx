# Sparkle Design Registry MCP 設定方法

次の設定を利用することで、Sparkle Designに準じた Sparkle Design Registry MCP サーバーを起動できます。<br />
このMCPサーバーはshadcn/uiに[2025年4月リリースされたMCP機能](https://ui.shadcn.com/docs/changelog#april-2025---mcp)を利用しています。詳細な活用法は[shadcnさんのX投稿](https://x.com/shadcn/status/1917597228513853603)を参照してください。<br />
en: You can start a Sparkle Design Registry MCP server that conforms to Sparkle Design by using the following configuration.<br />
This MCP server uses [the MCP function released by shadcn/ui in April 2025](https://ui.shadcn.com/docs/changelog#april-2025---mcp). For more information on how to use it, see [shadcn's X post](https://x.com/shadcn/status/1917597228513853603).

```json
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": ["-y", "shadcn@canary", "registry:mcp"],
      "env": {
        "REGISTRY_URL": "https://sparkle-design.vercel.app/r/registry.json"
      }
    }
  }
}
```
