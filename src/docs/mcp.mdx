# Sparkle Design Registry MCP 設定方法

次の設定を`components.json`に記述することで、Sparkle Designに準じた Sparkle Design Registry MCP サーバーを起動できます。<br />
このMCPサーバーはshadcn/uiのMCP機能を利用しています。詳細な導入方法は[公式のドキュメント](https://ui.shadcn.com/docs/mcp)を参照してください。<br />
en: You can start a Sparkle Design Registry MCP server that conforms to Sparkle Design by using the following configuration.<br />
This MCP server uses the MCP feature of shadcn/ui. For more information on how to use it, please refer to the [official documentation](https://ui.shadcn.com/docs/mcp).

```json
{
  "registries": {
    "@sparkle-design": "https://sparkle-design.vercel.app/r/{name}.json"
  }
}
```
