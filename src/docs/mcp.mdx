# shadcn MCP 設定方法

次の設定を利用することで、Sparkle Designに準じた shadcn MCP サーバーを起動できます。<br />
en: You can start a shadcn MCP server that conforms to Sparkle Design by using the following configuration.

```json
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": ["-y", "shadcn@canary", "registry:mcp"],
      "env": {
        "REGISTRY_URL": "https://sparkle-design.vercel.app/r/registry.json"
      }
    }
  }
}
```
