# Badge

バッジは状態、カテゴリ、またはその他の重要な情報を示すための小さなラベルコンポーネントです。

> **Server Component 互換**: このコンポーネントは Server Component からそのまま利用できます。

## インストール

```bash
npx shadcn@latest add https://sparkle-design.vercel.app/r/badge.json
```

または npm パッケージとして `@goodpatch/sparkle-design` をインストールしている場合はそのまま利用できます。

## 使い方

```tsx
<Badge size="md" variant="normal">5</Badge>
```
