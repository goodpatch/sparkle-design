# Icon

アイコンはテキスト情報を視覚表現でユーザーに提示するために使用するコンポーネントです。

> **Server Component 互換**: このコンポーネントは Server Component からそのまま利用できます。

## インストール

```bash
npx shadcn@latest add https://sparkle-design.goodpatch.com/r/icon.json
```

または npm パッケージとして `sparkle-design` をインストールしている場合はそのまま利用できます。

## 使い方

```tsx
<Icon icon="check" size={6} fill={true} />
```
