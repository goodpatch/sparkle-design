# Link

リンクは他のページやセクションへナビゲーションするためのコンポーネントです。

> **Server Component 互換**: このコンポーネントは Server Component からそのまま利用できます。

## インストール

```bash
npx shadcn@latest add https://sparkle-design.goodpatch.com/r/link.json
```

または npm パッケージとして `@goodpatch/sparkle-design` をインストールしている場合はそのまま利用できます。

## 使い方

```tsx
<Link href="https://example.com" isOpenInNew>
  外部サイトへのリンク
</Link>
```
