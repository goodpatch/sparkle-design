# InputPassword

インプットパスワードはパスワードの情報をユーザーから取得するために使用するコンポーネントです。

> **Server Component 互換**: このコンポーネントは Server Component からそのまま利用できます。

## インストール

```bash
npx shadcn@latest add https://sparkle-design.vercel.app/r/input-password.json
```

または npm パッケージとして `@goodpatch/sparkle-design` をインストールしている場合はそのまま利用できます。

## 使い方

```tsx
<InputPassword
  size="md"
  placeholder="パスワードを入力してください"
/>
```
