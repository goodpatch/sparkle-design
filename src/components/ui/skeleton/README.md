# Skeleton

スケルトンはユーザーにコンテンツが読込中であることを提示するために使用するコンポーネントです。

> **Server Component 互換**: このコンポーネントは Server Component からそのまま利用できます。

## インストール

```bash
npx shadcn@latest add https://sparkle-design.vercel.app/r/skeleton.json
```

または npm パッケージとして `@goodpatch/sparkle-design` をインストールしている場合はそのまま利用できます。

## 使い方

```tsx
<Skeleton className="h-4 w-[250px]" />
```
